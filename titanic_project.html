Titanic Data Analysis¶

In this report, analsis of passengers' data from the infamous ship Titanic is presented. Please see the Project Rubric for further details. This report contains a description of the data for some of Titanic's passengers and further down in the document we'll dive in to the analysis of the data using tools like Numpy and Pandas.

Preliminary Questions¶

We start our analysis by posing the questions below to help guide us in our analysis.

  1. How many passengers are in the data provided for analysis?
  2. How many passengers survived this tragedy?
  3. What was the average age of passengers on the Titanic?
  4. Is there any correlation with survival and passenger's gender?
  5. Is there any correlation with survival and any other passengers' attribute?

Data Dictionary¶

Variable Definition Key

survival     Survival                                                 0 = No, 1 = Yes

pclass       Ticket class                                             1 = 1st, 2 = 2nd, 3 = 3rd

sex          Sex

Age          Age in years

sibsp        number of siblings / spouses aboard the Titanic

parch        number of parents / children aboard the Titanic

ticket       Ticket number

fare         Passenger fare

cabin        Cabin number

embarked     Port of Embarkation                                       C = Cherbourg, Q = Queenstown, S = Southampton

Variable Notes

pclass: A proxy for socio-economic status (SES) 1st = Upper 2nd = Middle 3rd = Lower

age: Age is fractional if less than 1. If the age is estimated, is it in the form of xx.5

sibsp: The dataset defines family relations in this way... Sibling = brother, sister, stepbrother, stepsister Spouse = husband, wife (mistresses and fiancs were ignored)

parch: The dataset defines family relations in this way... Parent = mother, father Child = daughter, son, stepdaughter, stepson Some children travelled only with a nanny, therefore parch=0 for them.

Data Wranggling Phase¶

In the data wranggling phase of our data analysis, we give our data sets a closer look in case we need to do any Data Cleaning. In our case, the data sets look good except a few exceptions. The two instances that might create difficulty in our data analysis are the NaN values that appear in the Age and Cabin columns indicating missing values of age and cabin number for those passengers. Since we won't be focusing on specific cabins in our analysis, we won't clean up that column but since in this data analysis we will be examining whether age determines survival rate of Titanic passengers or not, we will clean up the age column of our csv data by removing passengers with no corresponding age data from our analysis.

After learning about personal details of the oldest person on the Titanic and his age, I am interested in exploring further if there is any correlation between survival and attributes like passengers' age, sex and ticket class. To do that, I use the function below.

In [17]:
#this function inputs two values and outputs the pearson's r value for these two input values

def correlation(x,y):
    std_x = (x - x.mean()) / x.std(ddof = 0)
    std_y = (y - y.mean()) / y.std(ddof = 0)
    return (std_x * std_y).mean()

Starting the correlation analysis with the correlation between Survival and passengers' Ticket class we come across a surprising result. The code below gave me a negative Pearson's r (-0.356) contrary to what I expected but a close look at the data set reveals that the output is correct. My interpretation of the result is what needs to be changed. Since Ticket class (Pclass) was represented in the data set by three different numbers in ascending order, it gave us a negative correlation but in reality there is a positive correlation between survival and Ticket class. Those with 1st class tickets are more likely to survive than those with second or third class tickets even though the correlation is not that strong (r = 0.356).
